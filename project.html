Government Python OpenCV

GUI Overhaul for a GigE Outdoor Situational Awareness Camera

Built using Python, OpenCV, PyQt6, and the Spinakker SDK

Developed enhancements for a wide field-of-view situational awareness system at NIWC Pacific using OpenCV and PyQt6. Implemented real-time image overlays, integrated multiple data streams, and built interactive GUI components including ROI selection tools. Solved coordinate scaling challenges between high-resolution camera input and display output, improving accuracy and usability of the system.
